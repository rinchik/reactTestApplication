## Simple app

### Stack
ExpressJS
React
CssNEXT
Webpack

**Note:** You can see that css is in files with *.scss*, SASS file extension only for IDE to properly parse Cssnext syntax. This app does not use SASS.
 
 
### Usage

Install dependencies:
```bash
$ npm i
```

Run bundle:
```bash
$ npm run bundle
```

Run dev server:
```bash
$ npm start
```
